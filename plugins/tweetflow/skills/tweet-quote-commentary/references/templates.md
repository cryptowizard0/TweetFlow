# Quote Templates

Use these templates as starting points, not rigid output formats. Compress or rewrite them to fit the source tweet.

## Template 1: Architecture Breakdown

```text
This is actually a <higher-order underlying issue>.

What matters:
- Layer 1: ...
- Layer 2: ...
- Layer 3: ...

Most people only see <surface layer>.

The real leverage is <your judgment>.
```

Use when the tweet describes a product decision, system problem, infra pattern, org issue, or execution tradeoff.

## Template 2: Trend Judgment

```text
The market is focusing on <surface signal>.

But the real shift is <core change>.

Why:
- ...
- ...
- ...

This will matter because <implication>.
```

Use when the tweet points at a launch, metric move, funding event, hiring wave, model release, policy change, or other directional signal.

## Editing Rules

- Delete placeholder brackets before posting.
- Collapse bullets if the post is getting too long.
- Prefer one strong implication over multiple weak ones.
- Keep the final quote tweet natural; it does not need to preserve the exact line breaks of the template.
